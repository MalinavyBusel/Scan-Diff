to install and runthe package, runin the dir with this repo:
    - docker-compose up

supported extensions:
    - jpg
    - jpeg
    - png

supported languages:
    - russian
    - english

usage:
    - open in your browser the address "localhost:5000/"
    - select image1 and image2
    - select your documents language
    - click "submit"
    - [optionally] repeat
